# Vision

## Vision Statement

Murmur is a real-time multi-agent chat platform where AI agents collaborate with humans and each other to accomplish complex goals. We are building toward a future where teams of specialized AI agents work alongside people in shared workspaces — communicating, producing artifacts, managing tasks, and maintaining persistent context across sessions.

## Problem Statement

Current AI chat interfaces are single-agent, single-conversation tools. Users cannot orchestrate multiple specialized agents, agents cannot talk to each other, and conversation context is lost between sessions. Teams working with AI lack a collaborative workspace where agents can be assembled, directed, and observed as they work together on multi-step goals.

This limits AI's utility for complex tasks that benefit from specialization (research + analysis + implementation), collaborative reasoning, and long-running autonomous execution.

## Target Audience

| Persona | Description | Key Needs |
|---------|-------------|-----------|
| AI-augmented developer | Developers building with AI agents who need a platform to prototype, test, and run multi-agent workflows | Real-time streaming, agent-to-agent communication, persistent history, extensible tool system |
| Jido framework user | Developers using the Jido agent framework who need a pre-built chat interface | Drop-in LiveView components, configuration-driven agent profiles, Hex package integration |
| AI researcher | Researchers exploring multi-agent collaboration patterns | Observable agent interactions, artifact production, task convergence tracking |
